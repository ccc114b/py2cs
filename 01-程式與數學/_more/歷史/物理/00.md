## 物理學史重要事件年表

| 年份 (約) | 事件/發現 | 貢獻者 | 領域/意義 |
| :---: | :--- | :--- | :--- |
| **公元前 400 年** | 提出**原子論**（物質由不可分割的微粒組成） | 德謨克里特 (Democritus) | 早期物質結構概念 |
| **公元前 3 世紀** | 確立**浮力原理**、槓桿原理 | 阿基米德 (Archimedes) | 靜力學、流體靜力學的奠基 |
| **1021 年** | 著作《光學之書》；奠定**光學**和**實驗方法** | 伊本·海薩姆 (Ibn al-Haytham) | 科學方法的先驅；光學研究的起點 |
| **1543 年** | 提出**日心說**（地球繞太陽運行） | 哥白尼 (Copernicus) | 宇宙觀的革命，將地球從宇宙中心移開 |
| **1609-1619 年** | 發現**行星運動三大定律** | 克卜勒 (Kepler) | 天體運動的精確數學描述 |
| **17 世紀初** | 發展**實驗與數學物理學**；研究落體運動和慣性 | 伽利略 (Galileo) | 「現代科學之父」；奠定動力學基礎 |
| **1687 年** | 發表《自然哲學的數學原理》；提出**萬有引力定律**和**三大運動定律** | 牛頓 (Newton) | 確立**古典力學**的宏偉體系 |
| **1801 年** | 進行**雙縫實驗**，支持光的**波動說** | 楊氏 (Thomas Young) | 光學研究的重要進展 |
| **1831 年** | 發現**電磁感應**現象 | 法拉第 (Michael Faraday) | 現代電力技術（發電機、變壓器）的基礎 |
| **1860 年** | 確立**統計力學**（描述大量分子行為） | 馬克士威 (Maxwell)、玻爾茲曼 (Boltzmann) | 連接微觀與宏觀現象的橋樑 |
| **1861-1864 年** | 建立**馬克士威方程組**，統一**電學、磁學和光學** | 馬克士威 (Maxwell) | 證明光是**電磁波**；經典物理學的高峰 |
| **1887 年** | 進行**邁克生-莫立實驗**（證明**光速不隨觀察者運動**而改變，為相對論鋪路） | 邁克生 (Michelson)、莫立 (Morley) | 推翻「以太」存在性的關鍵實驗 |
| **1895 年** | 發現**X 射線** | 倫琴 (Wilhelm Röntgen) | 醫療與工業應用；開啟對高能輻射的研究 |
| **1896 年** | 發現**放射性**現象 | 貝克勒爾 (Henri Becquerel) | 開啟原子核物理學研究 |
| **1897 年** | 發現**電子** | 湯姆森 (J.J. Thomson) | 發現第一個基本粒子；推翻原子不可分的觀念 |
| **1900 年** | 提出**量子假說**（能量是量子化的） | 普朗克 (Max Planck) | **量子力學**的誕生標誌 |
| **1905 年** | 提出**狹義相對論**（光速不變、時間膨脹、質能等價 $E=mc^2$） | 愛因斯坦 (Albert Einstein) | 徹底改變對時間、空間、質量和能量的理解 |
| **1911 年** | 發現**原子核**（原子的大部分質量集中在微小帶正電的核） | 拉塞福 (Ernest Rutherford) | 確立**原子結構**概念 |
| **1913 年** | 提出**波耳原子模型**（電子在分立能級上繞核運行） | 波耳 (Niels Bohr) | 將量子概念應用於原子結構，解釋原子光譜 |
| **1915 年** | 提出**廣義相對論**（引力是時空彎曲的表現） | 愛因斯坦 (Albert Einstein) | 現代宇宙學的基礎 |
| **1925-1927 年** | **量子力學**完備（提出薛丁格方程式、海森堡不確定性原理等） | 薛丁格 (Schrödinger)、海森堡 (Heisenberg) 等 | 描述微觀世界行為的基礎理論 |
| **1932 年** | 發現**中子** | 查兌克 (James Chadwick) | 完善原子核結構模型 |
| **1938 年** | 發現**核裂變**（重核分裂成較輕的核） | 哈恩 (Hahn)、斯特拉斯曼 (Strassmann) | 核能和核武器技術的基礎 |
| **1964 年** | 發現**宇宙微波背景輻射 (CMB)** | 彭齊亞斯 (Penzias)、威爾遜 (Wilson) | 有力證明**大爆炸理論**的正確性 |
| **1960s-1970s** | 建立**粒子物理學標準模型**（描述夸克、輕子與基本作用力） | 蓋爾曼 (Gell-Mann)、溫伯格 (Weinberg) 等 | 成功統一電磁力與弱核力（電弱統一理論） |
| **2012 年** | 發現**希格斯玻色子** | CERN (LHC 實驗) | 證實粒子獲得質量的機制，完成標準模型最後一塊拼圖 |
| **2015 年** | 首次直接探測到**引力波**（時空中的漣漪） | LIGO 團隊 | 證實廣義相對論預言；開啟**引力波天文學** |
