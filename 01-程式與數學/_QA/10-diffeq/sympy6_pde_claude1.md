============================================================
SymPy 偏微分方程求解例子
============================================================

1. 一維熱傳導方程: ∂u/∂t = k * ∂²u/∂x²
--------------------------------------------------
方程: Eq(Derivative(u1(x, t), t), k*Derivative(u1(x, t), (x, 2)))
使用分離變數法求解...
假設 u(x,t) = X(x)T(t)
分離變數後得到:
T'(t)/T(t) = k * X''(x)/X(x) = -λ

2. 一維波動方程: ∂²u/∂t² = c² * ∂²u/∂x²
--------------------------------------------------
方程: Eq(Derivative(u2(x, t), (t, 2)), c**2*Derivative(u2(x, t), (x, 2)))
d'Alembert 解: u(x,t) = f(x-ct) + g(x+ct)

3. 二維 Laplace 方程: ∂²u/∂x² + ∂²u/∂y² = 0
--------------------------------------------------
方程: Eq(Derivative(u3(x, y), (x, 2)) + Derivative(u3(x, y), (y, 2)), 0)
使用分離變數: u(x,y) = X(x)Y(y)
得到: X''(x)/X(x) + Y''(y)/Y(y) = 0

4. 一維擴散方程特解求解
--------------------------------------------------
方程: Eq(Derivative(u4(x, t), t), D*Derivative(u4(x, t), (x, 2)))
假設解: u(x,t) = exp(-λt) * sin(nπx/L)
其中 λ = D * (nπ/L)²

5. 求解具體的線性 PDE
--------------------------------------------------
方程: Eq(Derivative(u5(x, y), x) + Derivative(u5(x, y), y), 0)
解: Eq(u5(x, y), F(x - y))

6. 非齊次偏微分方程
--------------------------------------------------
方程: Eq(Derivative(u6(x, y), x) + Derivative(u6(x, y), y), x + y)
這類方程需要使用特徵線方法或其他數值方法求解

7. 驗證 PDE 解
--------------------------------------------------
測試函數: u = x**2 + y**2
Laplacian: ∇²u = 4
是否為調和函數: False

測試函數: u = exp(-t)*sin(x)
熱傳導方程檢驗: u_t - u_xx = 0
是否為解: True

============================================================
注意事項:
1. SymPy 對複雜 PDE 的求解能力有限
2. 許多實際問題需要數值方法
3. 邊界條件和初始條件對解的唯一性很重要
4. 分離變數法是常用的解析方法
