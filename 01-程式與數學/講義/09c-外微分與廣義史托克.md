## 基本 k 形式

以下都是針對流形局部坐標系而言

1. 基本 1 形式:（偏微分） 就是坐標系下的微分 $dx^i$，其中 $i = 1, \dots, n$。 

    說明：任何一個 1 階微分形式 $\omega$ 都可以寫成這些基本 1 形式的線性組合： $$\omega = F_x \, dx + F_y \, dy + F_z \, dz$$


2. 基本 $k$ 形式: (k 個偏微分的外積)

    $$\text{基本 } k \text{ 形式} = dx^{i_1} \wedge dx^{i_2} \wedge \dots \wedge dx^{i_k}$$

3. k 階微分形式 ( $\omega$ ) ： 

    $$\omega = \sum_{1 \le i_1 < \dots < i_k \le n} f_{i_1 \dots i_k} \, dx^{i_1} \wedge \dots \wedge dx^{i_k}$$

    直覺意義：k 維相關的物理量

    一維：線的物理量（路徑或線段上的某種量，動量，...）
    二維：面的物理量 (面積，通量...)
    三維：體的物理量 (體積，密度...)

    $k$ 階微分形式是微小的、有向的 $k$ 維感測器，用來測量空間中某個場的特定屬性。

    具有
    
    1. 反對稱性: $dx \wedge dy = -dy \wedge dx$
    2. 協變性:坐標系變換不影響積分值

4. 外微分 ( $d\omega$ ) :

    $$d\omega = \sum_{1 \le i_1 < \dots < i_k \le n} df_{i_1 \dots i_k} \wedge dx^{i_1} \wedge \dots \wedge dx^{i_k}$$

    其中

    $$df_{i_1 \dots i_k} = \sum_{j=1}^n \frac{\partial f_{i_1 \dots i_k}}{\partial x^j} \, dx^j$$

    外微分 $d$ 將 $k$ 階形式 $\omega$ 變成 $k+1$ 階形式 $d\omega$。這在幾何上代表：$d\omega$ 測量了 $\omega$ 在某個微小 $k$ 維體積的邊界上累積的「流動」或「變化」。

5. 廣義史托克定理:

    $$\int_{M} d\omega = \int_{\partial M} \omega$$

    1. 一個場（微分形式 $\omega$）在整個區域 $M$ 內部的總累積變化量（$d\omega$ 的積分），完全由該場在區域 $M$ 的邊界 $\partial M$ 上的值（$\omega$ 的積分）所決定。

    2. 外微分 $d$ 總是將 $k$ 階形式（在 $\partial M$ 上積分）提升為 $k+1$ 階形式（在 $M$ 上積分）。
        * 1 階 (線) $\to$ 2 階 (面積): 線積分（邊界效應）等於面積積分（內部渦流）。
        * 2 階 (面積) $\to$ 3 階 (體積): 面積分（邊界通量）等於體積積分（內部源彙）。

    * 物理範例：守恆定律
        * 如果一個場 $\omega$ 是閉合的 (closed)，即 $d\omega = 0$，則斯托克斯定理給出：$$\int_{M} 0 = \int_{\partial M} \omega \quad \implies \quad \int_{\partial M} \omega = 0$$
        * 這意味著：如果場 $\omega$ 在內部沒有源（即 $d\omega=0$），那麼它在任何閉合邊界 $\partial M$ 上的淨累積或淨流動總和必須為零
        * (能量守恆，質量守恆，質能守恆 ...)
        * 例如，如果一個流體是無源的 ($\text{div} \mathbf{F} = 0$)，那麼流經任何閉合曲面（邊界）的淨通量必須為零。這就是流體的質量守恆。
        * 如果一個力場 $\mathbf{F}$ 是保守的 ($\text{curl} \mathbf{F} = \mathbf{0}$)，則沿任何閉合路徑的功（環流量）為零。這就是能量守恆。

6. 冪零性 $d^2 = d(d\omega) = 0$ ： (直覺：邊界沒有邊界)

    為何連續兩次外微分後必須為 0 ?

    1. 任何一個流形（無論是線段、曲面或體積）的邊界本身是沒有邊界的。
        * 在拓撲學中，我們形式化地定義：任何流形的邊界的邊界為空：$$\partial(\partial M) = \emptyset$$
    2. 數學上，根據 數學：二次導數的對稱性
        * 針對 0 形式 ($d(df)=0$)：是混合二階偏導數對稱性與外積反交換性相抵消的結果。
        * 針對 $k \ge 1$ 形式 ($d(d\omega)=0$)：主要原因是外積的冪零性 ($dx^i \wedge dx^i = 0$)，它保證了在第二次微分 $d(d\omega)$ 的過程中，新產生的 $dx^j$ 必然會在更高階的外積中重複出現，從而使整個項變為零。
